# rabbit
Example idiomatic erlang binary terms over rabbitmq


## HOWTO
You will need docker (preferably ver 1.4) 
You will need cmake and gcc

### Run
```
./setup.sh
```

Assuming by some miracle that worked... 

### Run
```
./run.sh
```

After about 6 seconds you should have seen 100 messages flash by. Congratulations.
